Index.html is special

Home > Index pages >

The file index.html is often a special file and can be omitted from URL. For example: 

<a href="http://eg-146.cpjobling.org.uk/">EG-146 Demo Web Site</>
is actually HTML markup for a link to http://eg-146.cpjobling.org.uk//index.html.

It's a good idea to put an index.html file into each folder. Otherwise, the web server may show the actual files! Like this: projects/.
