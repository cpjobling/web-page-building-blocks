Absolute Paths

These are the files linked using absolute paths linked to the document root /. I have used the following notation to define the type of resource:

  • [d] is a directory (or folder)
  • [w] is a web page (.html)
  • [t] is a plain text file (.txt)
  • [g] is a graphic file (.png, .gif or .jpeg)
  • [s] is a CSS style file (.css)

The relative indentation indicates the directory structure used on the website. Hover over the link and observe the URL displayed in your browser's status bar to understand how resources map to URLs.

Relative Paths

These are the files linked using paths relative to the document root /. As before, hover over the link and observe the URL displayed in your browser's status bar to understand how resources map to URLs. Note that the browser should display exactly the same URL for a given resource whether you use absolute or relative paths for your links. The browser is computing these paths from the rules given in the lecture and described in the web pages on this web site.
