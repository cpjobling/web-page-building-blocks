Homework Exercises: Website Addressing and URLs

Home > Homework >

  1. Draw a map of this web site on paper.
  2. Write down the complete URL of every page and image on the site and hence or otherwise, tabulate the absolute paths of all the objects on the site.
  3. From the file projects/projecta.html write down the URL to all other web pages and images accessible on this web site.
  4. View the source to investigate the structure of the navigation panel in these pages. Why are absolute paths preferred to relative paths in navigation panels?
  5. The images used in the logo for this website uses relative paths is this a good design choice? If not, why not?
  6. Create an index file for the projects folder.
  7. A new section news is to be added to the site. It will contain a folder for news items arranged by year, month, and day. Assuming that news for for 7th March 2011 will be found at URL http://eg-146.cpjobling.org.uk/news/2011/march/07/index.html). Design a navigation panel to access the current week's news. You don't need to use a calendar but that would be one way to do it.
  8. If each news item was to have an accompanying picture, where would you put it? Justify your choice.
  9. Imagine that each day's news is arranged on a single page under headings ordered by the time of posting (to the nearest hour). For example, news at 10.00 am, 11.00 am, etc. Suggest a way of marking up the news items using <h2> tags with the posting time used for an id attribute. Write down the full URL of news items reported after 12.00 noon on Tuesday 8th May, 2012.
  10. Examine the contents of the style folder in preparation for the CSS lectures to come.
