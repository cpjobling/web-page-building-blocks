Web Page Building Blocks and Web Site Design

Presentation for Lecture 14 of the 2010-2011 delivery of EG-146. Converted to showoff format from PowerPoint.

Implemented using Showoff http://github.com/schacon/showoff

Instructions
  $ gem install showoff
  $ git clone git://github.com/scottbale/JavaScript-Function-Idioms.git
  $ cd JavaScript-Function-Idioms
  $ showoff serve

Then you should be able to go to http://localhost:9090 to view the
presentation.
